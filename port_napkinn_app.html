Detalhes do projeto

Aplicativo Napkinn Health

Aplicativo voltado para os colaboradores de empresas que possuam algum plano de saúde. A missão é analisar e mapear a saúde dos usuários para gerar um plano de gamificação personalizado, criando uma iniciativa de mudança de hábitos para um melhor estilo de vida.

 TECNOLOGIAS UTILIZADAS

Ionic
Ionic é um framework para criação de aplicativos híbridos. Possui como linguagem base o Typescript
e também permite a manipulação do HTML e CSS para o front-end além de componentes próprios para acessar recursos do smartphone.
NodeJS
NodeJS juntamente ao Express compõe as ferramentas utilizadas para criação da REST API. A linguagem utilizada é a Javascript.
MySQL
MySQL é o sistema gerenciador de banco de dados desse projeto, baseado na linguagem SQL.
Docker
Orquestração de containers para o desenvolvimento do software.
Python
Linguagem de programação utilizada para criação de serviços e rotinas em execução no backend.
MongoDB
Sistema gerenciador de banco de dados NoSQL, utilizado para lidar com a alta demanda de dados.
