Linha de cuidado odontológica - Sistema WEB

A Linha de cuidado odontológica é um software que visa mudar a forma como os tratamentos odontológicos acontecem, com foco total no paciente e em suas necessidades. As funcionalidades suportam o paciente de ponta a ponta, desde o seu diagnóstico até o plano de tratamento. O mesmo acontece com o profissional de saúde, facilitando sua vida na realização e aprovação de procedimentos por parte da operadora.

Empresa: NK Tech

Open Source: Não

 TECNOLOGIAS UTILIZADAS

Angular
Angular é um framework robusto desenvolvido pela Google para criação de sistemas multiplataforma.
Possui como linguagem base o Typescript e também permite a manipulação do HTML e CSS para o front-end.
NodeJS
NodeJS juntamente ao Express compõe as ferramentas utilizadas para criação da REST API. A linguagem utilizada é a Javascript.
MySQL
MySQL é o sistema gerenciador de banco de dados desse projeto, baseado na linguagem SQL.
Docker
Orquestração de containers para o desenvolvimento do software.
Python
linguagem de programação utilizada para criação de serviços internos.
