Detalhes do projeto

Classificação de produtos com Processamento de Linguagem Natural (PLN)

Esse projeto serviu como base para estruturação de um grande conjunto de dados contendo milhares de produtos de supermercado diferentes. Utilizando apenas as informações presentes na embalagem dos produtos, como nome e descrição, foi criado dois modelos de inteligência artificial baseado em redes neurais convolucionais que atuam em conjunto, para classificar se o produto é ou não alimentício e qual a sua categoria, para ser organizado no banco de dados.
