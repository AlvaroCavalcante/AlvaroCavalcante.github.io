Detecção de Ferrugem e Bicho Mineiro na folha do Café

Esse foi um trabalho de conclusão de curso (TCC) do curso de Big Data no Agronegócio pela FATEC. Foram coletados mais de 500 folhas de café contendo as fitopatologias da ferrugem e/ou bicho mineiro. A partir daí, foi criado um algoritmo de inteligência artificial baseado em redes neurais convolucionais para detectar os objetos na imagem. Além disso, um aplicativo - Coffee Recognize - também foi desenvolvido para servir de interface.

 TECNOLOGIAS UTILIZADAS

TensorFlow
Plataforma utilizada para a criação de toda a arquitetura de detecção de objetos e treinamento do modelo de IA.
Ionic
Ionic é um framework para criação de aplicativos híbridos. Possui como linguagem base o Typescript
e também permite a manipulação do HTML e CSS para o front-end além de componentes próprios para acessar recursos do smartphone.
NodeJS
NodeJS juntamente ao Express compõe as ferramentas utilizadas para criação da REST API. A linguagem utilizada é a Javascript.
Python
Linguagem de programação utilizada para criação da pipeline de treinamento e de inferência do modelo.
